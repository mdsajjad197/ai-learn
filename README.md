#
AI Learning Companion

An AI-powered learning platform that helps you study smarter, not harder.

## Features
- **Document Analysis**: Upload PDFs and get instant summaries.
- **AI Chat**: Chat with your documents to ask specific questions.
- **Flashcards**: Automatically generate flashcards from your study materials.
- **Quizzes**: Test your knowledge with AI-generated quizzes.
- **Dashboard**: Track your learning progress with detailed statistics.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express, MongoDB
- AI: Integration with OpenRouter/LLMs

## Getting Started
1. Install dependencies: `npm install`
2. Start backend: `cd backend && npm start`
3. Start frontend: `cd frontend && npm run dev`
